# React-Quiz App (Course project)

A quiz application built to practice component state, mapping lists and scoring logic in React.

## Key features

- Multiple-choice questions from a local JSON file
- Track score and progress
- Simple feedback per question

## Tech

- React (Create React App)

## Run locally

1. Install dependencies: `npm install`
2. Start dev server: `npm start`

## Notes

- Questions are stored in `questions.json` in the `src` folder. To add or change questions, update that file.
